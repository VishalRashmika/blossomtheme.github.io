    Installation

    1. Open File Explorer and navigate to: %LOCALAPPDATA%\SumatraPDF\

    2. Create a backup of SumatraPDF-settings.txt:

    3. Edit SumatraPDF-settings.txt:

    4. Select and delete lines 3 through 48

    5. Open your downloaded theme file Copy its contents

    6, Paste into SumatraPDF-settings.txt where you deleted the lines Save and close the file

    1. If SumatraPDF is open, close it Reopen SumatraPDF to see the new theme

    About BlossomTheme

    A color theme inspired by Japan's vibrant urban environment, blending bright neon tones and metallic elements to evoke a modern and energetic visual atmosphere.
